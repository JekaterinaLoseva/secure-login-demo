IAM Authentication Demo: Behind the Scenes

Insecure Login Vulnerable

Passwords are sent and stored in Plain Text. No protection.

💾 Database Storage View (Plain Text):
SELECT * FROM users;
[{ id: 1, user: "test_user", pass: "Password123!" }]

Secure Login Best Practice

Passwords are salted and cryptographically hashed (SHA-256).

💾 Database Storage View (Salted Hash):
SELECT * FROM users;
[{ id: 1, user: "test_user", salt: "8f3b2a1c...", hash: "e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855" }]
