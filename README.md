# Secure Login Demo (IAM Basics)

This repository serves as a code prototype for the CyberStart Final Project. It highlights the critical difference between insecure and secure password management in identity systems.

## Project Structure
- **InsecureAuth.java**: Demonstrates the dangerous approach of storing and verifying passwords in plain text.
- **SecureAuth.java**: Demonstrates industry best practices by utilizing unique cryptographic Salting and SHA-256 Hashing to protect credentials.
